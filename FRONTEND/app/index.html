InSoLiTo is a graph-based network of the co-usage of research software from the tools collection hosted by OpenEBench.

Co-usage is understood as being cited in the same cientific publications.

Search you favourite research software to see how it relates with the other tools inside the community. If you search by topic, it will display the software related to the given branch in bioinformatics.
